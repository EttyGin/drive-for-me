שינוי צבעים ב-Z2JH דרך values.yaml

ב-Z2JH, ה-Frontend משתמש ב-JupyterHub UI, וניתן להתאים אישית את ה-CSS על ידי הוספת קובץ סטייל מותאם אישית, או על ידי שימוש בהגדרות hub.extraConfig.

אם את רוצה לשנות את הצבע הכתום לקרמל, ניתן לבצע זאת כך:

  1. שימוש ב-Custom CSS דרך hub.extraConfig
    ניתן להוסיף קובץ CSS מותאם אישית שיחליף את הצבע הכתום בצבע קרמל:

yaml
hub:
  extraConfig:
    customCSS: |
      import os
      with open("/etc/jupyterhub/custom/custom.css", "w") as f:
          f.write("""
          :root {
              --jp-brand-color1: #c19a6b !important; /* צבע קרמל */
              --jp-layout-color1: #f5f5dc !important; /* רקע בהיר */
          }
          """)

  1. הוספת קובץ CSS מותאם אישית דרך Persistent Volume
    ניתן להעלות קובץ custom.css ולצרף אותו לקונטיינר של ה-Hub:

yaml
singleuser:
  extraFiles:
    custom.css:
      mountPath: /etc/jupyterhub/custom/custom.css
      data: |
        :root {
            --jp-brand-color1: #c19a6b !important;
            --jp-layout-color1: #f5f5dc !important;
        }

לאחר השינויים האלו, יש להריץ עדכון ל-Helm:

sh
helm upgrade --install jhub jupyterhub/jupyterhub --values values.yaml

אם את רוצה שינוי צבעים נוסף ברמת ה-Frontend, ניתן גם לעדכן את JupyterLab דרך ה-UI עצמו, או לשלב ערכות נושא מותאמות אישית.
